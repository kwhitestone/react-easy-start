## React Easy Start
A lightweight ant-design-pro based on create-react-app.
## Installation
```
git clone https://github.com/sunnut/react-easy-start.git
cd react-easy-start
npm install
npm run mock
npm start
```  
## Dependencies
* create-react-app
* React 17.0.1    --hooks
* Redux           --state management
* React-Router-V4 --page management
* Ant Design      --UI components
* React I18NExt   --I18N
* Restify         --Mock Server
